## 介绍

`@ipare/mva` 是 `Ipare` 的 MVA 框架

- 支持 RESTful 规范
- 根据文件系统映射访问路径，彻底解耦无关联功能
- 轻量化，高可扩展性
- 移除 controller 层，灵活性更高

## 安装

```
npm install @ipare/mva
```
