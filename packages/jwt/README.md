## 介绍

`@sfajs/jwt` 是 `sfajs` 的 jwt 工具插件

基于 [jsonwebtoken](https://github.com/auth0/node-jsonwebtoken) 和 [@sfajs/inject](https://github.com/sfajs/inject)

## 安装

```
npm install @sfajs/jwt
```

## 开始使用

请访问 <https://sfajs.com>
