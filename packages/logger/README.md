## 介绍

`@ipare/logger` 是 `Ipare` 的日志插件

基于 [winston](https://github.com/winstonjs/winston)

通过依赖注入 [inject](https://github.com/ipare/ipare) 的方式使用

## 安装

```
npm install @ipare/logger
```

请访问 <https://ipare.org>
