## 介绍

`@sfajs/static` 是 `sfajs` 的静态资源插件

- 能够返回静态资源
- 能够匹配单个文件或整个文件夹
- 可以托管静态网站

## 安装

```
npm install @sfajs/static
```

请访问 <https://sfajs.com>
