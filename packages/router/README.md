## 介绍

`@sfajs/router` 是 `sfajs` 的路由插件

- 支持 RESTful 规范
- 根据文件系统映射访问路径，彻底解耦无关联功能
- 按需加载，提升请求速度
- 轻量化，高可扩展性
- 移除 controller 层，灵活性更高
- 预编译路由，提升启动速度和响应速度

## 安装

```
npm install @sfajs/router
```
