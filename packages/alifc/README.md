## 介绍

`@ipare/alifc` 是 `Ipare` 的阿里云云函数计算数托管环境

可以将 Ipare 托管到阿里云函数计算，有效提升云函数响应速度

## 安装

```
npm install @ipare/alifc
```

## 开始使用

请访问 <https://ipare.org>
