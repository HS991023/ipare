## 介绍

`@sfajs/http` 是 `sfajs` 的 http 托管环境

让 sfa 能够运行于 http(s) 服务

你可以随时将 sfa 云函数简单改为 nginx 托管的 sfa http 服务，当然也可以将 nginx 托管的 sfa http 服务简单改为 sfa 云函数

## 安装

```
npm install @sfajs/http
```

请访问 <https://sfajs.com>
